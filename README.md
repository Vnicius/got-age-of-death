# GoT - Age of Death

Find out your age of death in Game of Thrones based on your name.

## Dependencies

- Python 3.x

### Anaconda

```sh
  $ conda env create -n got-age-of-death -f environment.yml
  $ conda activate got-age-of-death
```

### Pip

```sh
  $ pip install -r requirements.txt
```

## Usage

```sh
  $ jupyter notebook
```
